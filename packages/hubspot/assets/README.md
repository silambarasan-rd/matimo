# HubSpot assets

This folder holds images and other static assets for the HubSpot provider package.

Available assets:

- **hubspot.png**: Official HubSpot logo (downloaded from HubSpot brand assets).
- **hubspot.svg**: Official HubSpot SVG logo (downloaded from HubSpot brand assets).
- **hubspot-icon.png**: Small icon (PNG) for display in UI — can be created from SVG if desired.
 - **hubspot-o.svg**: Isolated HubSpot mark (the orange "O") suitable as an icon.
 - **hubspot-icon.png**: Small icon (PNG) for display in UI — can be created from SVG if desired.
